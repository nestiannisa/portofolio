Codi Health
Codi Health adalah aplikasi e-commerce yang saya kerjakan bersama rekan rekan saya. Codi health ini sendiri merupakan e-commerce dalam penjualan obat-obatan. dalam projek ini saya bertugas dalam pembuatan backend. Untuk bagian Frontend saya menggunakan:
  • Reactjs + Redux
  • Axios
  • Sweet Alert
  • React Bootstrap
  • React Router Dom
Sedangkan untuk bagian Backend saya menggunakan:
  • Nodejs
  • Sequelize
  • Postgresql
  • Bcrypt
  • Cors
  • Dotenv
  • Jsonwebtoken
  • express
  • multer
