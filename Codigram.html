Codigram
Codigram adalah aplikasi media sosial yang digunakan sebagai wadah menampung aspirasi dan bertukar fikiran tentang perkembangan yang ada di dunia IT. Untuk bagian Frontend saya menggunakan:
  • Reactjs + Redux
  • Axios
  • Sweet Alert
  • React Bootstrap
  • React Router Dom
Sedangkan untuk bagian Backend saya menggunakan:
  • Nodejs
  • Sequelize
  • Postgresql
  • Bcrypt
  • Cors
  • Dotenv
  • Jsonwebtoken
  • express
  • multer
